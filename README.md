# Transfer potential of F-like plasmids differ by animal environment

This repository contains code and data to reproduce the results in "[Transfer potential of F-like plasmids differ by animal environment](https://doi.org/10.64898/2026.02.02.703319)" by Sneha Sundar, Sebastian Bonhoeffer and Jana S. Huisman. 

- code:  
    contains code to reproduce figures and tables
- data:  
  contains data files used to reproduce figures and tables
- find-genes-workflow:  
  Snakemake pipeline to reproduce bioinformatic analyses. Large files (such as genomes and annotation files) are not included in this repository. Accession numbers for the genomes used have been listed.
